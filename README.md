## 目的

このフォルダには、**Complex Multilayered Abuse（CMA）**研究の基盤となる3つの文書を「最新版」として固定しています。

これにより、将来的に新しい文章（X投稿、学会要旨、論文、著者紹介、取材回答など）を作成する際に、**定義・理論的立ち位置・表現の整合性**を保ち、文書が枝分かれしたり、矛盾が生じたりするのを防ぐことができます。

「新しい理論を追加する」のではなく、まずはこの3文書をしっかり固定し、**研究の顔**として安定させることを優先します。

---

## 固定されている3文書

| ファイル名                        | 内容                     | 主な用途                          |
|-----------------------------------|--------------------------|-----------------------------------|
| `CMA_01_Theory_Main.md`           | CMA本体の理論説明（詳細） | 理論の根幹、論文・発表のベース     |
| `CMA_02_Summary_200chars.md`      | CMAの200字程度の要約     | 概要紹介、Researchmap、プロフィール |
| `CMA_03_Position_Researchmap.md`  | 研究者としての立ち位置・方法論 | Researchmap、学会、著者紹介、取材対応 |

## 推奨運用フロー

1. 新しい文章を作成する前・作成中に、この3文書を必ず読み返す。
2. 用語の定義（CMAとは何か）、理論の位置づけ（診断ではなく補完的フレームワーク）、自身の立ち位置（当事者研究×質的統合分析など）が一致しているか確認。
3. 必要に応じて微調整するが、**大きな変更はバージョンアップ**して固定。
4. このフォルダ全体をバックアップし、Researchmapの「研究内容」や共有フォルダに置いておく。
