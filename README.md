## 目的

このフォルダには、**Complex Multilayered Abuse（CMA）**研究の基盤となる3つの文書を「最新版」として固定しています。

これにより、将来的に新しい文章（X投稿、学会要旨、論文、著者紹介、取材回答など）を作成する際に、**定義・理論的立ち位置・表現の整合性**を保ち、文書が枝分かれしたり、矛盾が生じたりするのを防ぐことができます。

「新しい理論を追加する」のではなく、まずはこの3文書をしっかり固定し、**研究の顔**として安定させることを優先します。

## Complex Multilayered Abuse（CMA）とは

CMA（Complex Multilayered Abuse）は、複数の虐待経験が発達過程および環境要因と相互作用しながら維持される構造を記述するための概念モデルである。

本モデルは、虐待を個別の出来事や症状のみとして捉えるのではなく、複数の虐待経験、対人関係、環境条件、支援へのアクセス、認知的・身体的適応などが相互に関連しながら持続する構造として理解することを目的とする。

CMAは診断基準や治療モデルではなく、虐待経験の背景に存在する構造的要因を整理・記述するための補完的フレームワークとして位置づけられる。

本モデルは、長期的な自己観察記録、行政・医療記録、および既存の虐待・トラウマ研究との比較検討を通じて構築され、支援不応答（Institutional/Environmental Non-Responsiveness）や閉鎖環境の影響を含めた構造的理解を試みている。

# CMA立ち位置（Researchmap・学会向け） — 最新版 v1.0

**作成日**: 2026年6月3日  
**作成者**: 中村達希

---

中村達希は、当事者研究、障害学、社会福祉学、トラウマ研究を横断する立場から、虐待経験および支援不応答の構造分析に取り組んでいる。

長期的な自己観察記録（lived experience）と行政・医療記録を統合し、質的統合分析を用いて虐待経験の維持構造や支援への到達困難性を検討している。

また、複数の虐待経験が発達過程や環境要因と相互作用しながら維持される構造を記述する概念モデルとして、Complex Multilayered Abuse（CMA）を提案し、その理論的および実践的意義について研究を進めている。
